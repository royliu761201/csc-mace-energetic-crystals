# CSC-MACE energetic molecular crystals submission archive

This repository contains the manuscript source, compiled PDF, and curated submission package for:

**Charge-self-consistent equivariant potentials for energetic molecular crystals: static gains and nonequilibrium limits**

## Contents

- `paper/`: LaTeX source, bibliography, figures, and compiled manuscript PDF.
- `submission/final_upload_package/`: finalized submission-facing files, including the manuscript PDF, cover letter PDF, and supplementary evidence notes.
